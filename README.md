# Reminder Bot

Reminder Bot adalah bot Telegram yang membantu pengguna membuat dan menerima pengingat secara otomatis sesuai waktu yang telah ditentukan.

## Fitur

- Menambahkan pengingat
- Melihat daftar pengingat
- Menghapus pengingat
- Notifikasi otomatis sesuai jadwal

## Teknologi

- Python
- Telegram Bot API
- SQLite
- APScheduler

## Instalasi

Clone repository:

```bash
git clone https://github.com/gadizafauzi/reminder_bot.git
cd reminder_bot
```

Install dependensi:

```bash
pip install -r requirements.txt
```

Jalankan aplikasi:

```bash
python bot.py
```

## Cara Penggunaan

- `/start` → Memulai bot
- `/remind` → Menambahkan pengingat
- `/list` → Melihat daftar pengingat
- `/delete` → Menghapus pengingat

## Tujuan Proyek

Proyek ini dibuat untuk mempelajari pengembangan bot Telegram, pengelolaan database, dan sistem penjadwalan otomatis menggunakan Python.
